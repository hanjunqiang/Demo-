# DayDayNews
仿网易新闻客户端，实现新闻浏览，视频播放，抓取百度图片，瀑布流显示

###修改了视频显示方式
_点击当前cell播放视频在当前cell上，监听屏幕转动，当屏幕转动的时候，视频自动横屏全屏播放，当屏幕为正的时候，播放在当前cell上<br />
_增加了活动指示器，采取搜狐视频活动指示器
